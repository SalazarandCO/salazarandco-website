# rodrigogana.com

Personal site for Rodrigo Gana — mirrors the salazarandco.com Nichols-template build.

## Files
- `index.html` — Main single-page site
- `contact.html` — Contact / inquiry page
- `images/` — Headshot, project images, press images
- `vercel.json` — Routing config

## Deploy
This folder is self-contained. To deploy:
1. In Vercel, create a new project pointing at this folder (or set the **Root Directory** to `rodrigogana/` if deploying the parent repo).
2. Connect the `rodrigogana.com` domain to the project.
3. Vercel will auto-serve `index.html` at the root.

## Adding the headshot
Drop a JPG named `rodrigo-headshot.jpg` into `images/` — the editorial pane on the homepage will use it automatically. Recommended: portrait crop, 800px+ wide, black-and-white or color (the site reads both well).
